# 🚖 دليل نشر WinRak على المتاجر

---

## ⚡ الخطوة 0 — التسجيل في Expo (مجاني)

```cmd
eas login
```
سيطلب منك إنشاء حساب على expo.dev (مجاني) أو تسجيل الدخول.

---

## 📱 جزء أول: Google Play Store (أندرويد)

### المتطلبات:
- ✅ حساب Google Play Developer: **25$ مرة واحدة**
- 🔗 سجّل هنا: https://play.google.com/console

### الخطوات:

#### 1. اربط المشروع بـ Expo
```cmd
cd C:\winrak\apps\mobile
eas project:init
```

#### 2. ابنِ APK للاختبار الداخلي أولاً
```cmd
eas build --platform android --profile preview
```
⏳ ينتظر على خوادم Expo (10-15 دقيقة)
بعدها يعطيك **رابط تحميل APK** — شاركه مع المختبرين مباشرة!

#### 3. ابنِ AAB للنشر الرسمي
```cmd
eas build --platform android --profile production
```

#### 4. ارفع على Google Play Console
- افتح: https://play.google.com/console
- أنشئ تطبيق جديد → اسمه "WinRak"
- اذهب لـ: Testing → Internal testing
- ارفع ملف AAB
- أضف المختبرين بإيميلاتهم (حتى 100 شخص)
- **النشر فوري بدون مراجعة!** ✅

#### 5. أو ارفع تلقائياً
```cmd
eas submit --platform android --latest
```

## 🍎 جزء ثاني: Apple App Store (iOS)

### المتطلبات:
- ✅ حساب Apple Developer: **99$/سنة**
- 🔗 سجّل هنا: https://developer.apple.com/programs

### الخطوات:

#### 1. ابنِ IPA
```cmd
eas build --platform ios --profile production
```

#### 2. ارفع على TestFlight
- افتح: https://appstoreconnect.apple.com
- أنشئ تطبيق جديد → Bundle ID: `dz.winrak.app`
- ارفع الـ IPA عبر TestFlight
- أضف المختبرين (حتى **10,000 مختبر**!)
- مراجعة بسيطة (24-48 ساعة)

#### 3. أو ارفع تلقائياً
```cmd
eas submit --platform ios --latest
```

## 🚀 أسرع طريقة للبدء (اليوم!)

```cmd
cd C:\winrak\apps\mobile

# 1. سجّل في Expo
eas login

# 2. ابنِ APK مجاناً (للاختبار الفوري)
eas build --platform android --profile preview

# 3. حمّل APK على هاتفك وجرّبه!
```

**لا تحتاج أي متجر للخطوة 2 — فقط APK تثبّته مباشرة!**
